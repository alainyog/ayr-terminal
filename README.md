================

The source of ayrtonaraujo.net website.

## Quick Start

 Install dependencies:

    $ npm install

 Start the server:

    $ node app

## Running Tests

To run the test suite first invoke the following command within the repo, installing the development dependencies:

    $ npm install

then run the tests:

    $ make test
